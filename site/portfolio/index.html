What's built

Kinetika isn't a slide deck — it's running. Here's the ecosystem, layer by layer, with honest status on each.

Live

The Homelab

A two-node Proxmox cluster running the whole stack: reverse-proxy gateway, SSO, internal DNS, monitoring and metrics, NAS + backup tiers, media services, and automation. Public services reach the world through a Cloudflare Tunnel — no exposed home IP.

Proxmox · NPM · Authelia · n8n · Nextcloud · Immich · Jellyfin · Grafana

Live

Kinetika HQ

A single static launcher for every internal tool, served by Caddy behind SSO. The card catalog auto-derives public routes; private endpoints stay in an untracked local config. It's the front door to the homelab.

Static · Caddy · SSO · self-hosted

Live

Kost HQ

Property management run over WhatsApp using Baileys — tenants, payments, and operations handled in the channel people already use. A v2 on the AX layer is on the roadmap.

Node.js · Baileys · WhatsApp
